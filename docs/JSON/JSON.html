JSON

Parse and Stringify

Valid JSON

  • is a name value pair
  • it's keys must be strings
    •          For example...
            { "class":"CS261" }
         NOT
            { class: "CS261" }
  • it's value types must be; string, number, object, array, boolean, or null
  • it's keys and values must be in double quotes, not single
    •          For example...
            { "class":"CS261" }
         NOT
            { 'class': 'CS261' }

Avoid using functions in JSON.

JSON vs XML

JSON Objects

         {
            "student": { "ID":"8675309", "name":"Jenny", "age":"38" }
         }

JSON Arrays

         {
            "students": [ "Jenny", "Tommy", "Tutone" ]
         }

For the following examples we will use this JSON:

      myJSON = { 
         "song":"Girls Life You",
         "artist":"Maroon 5",
         "billboard":1, 
         "sameArtist": {
            "num1":"Moves Like Jagger",
            "num2":"One More Night".
            "num3":"Payphone"
            } 
         }

Accessing inside JSON

         var mySong = myJSON.song;
      or...
         var mySong = myJSON["song"];

Looping through JSON

         for (x in myJSON) {
            var chartTop += x;
         }
      or...
         for (x in myJSON) {
            var chartTop += myJSON[x];
         }

Nested JSON

         var mySong = myJSON.sameArtist.num1;
      or...
         var mySong = myJSON.sameArtist["num1"];

Modifying or Deleting JSON

         myJSON.sameArtist.num1 = "Payphone";
      or...
         myJSON.sameArtist["num1"] = "Payphone";

Use 

delete
to remove things from JSON

         delete myJSON.sameArtist.num3;
