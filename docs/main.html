For Loop

      for (statement 1; statement 2; statement 3) {
         code here...
      }

Statement 1 is done before entering the { }. It is usually used to set a variable to count the number of loops. Such as: 

      var i = 0;

Statement 2 is the conditional to determine in the code inside the { } will be executed. For example: 

      i < 5;

Statement 3 is executed at the end of each loop through. It is often used to iterate the Statement 1. For example: 

      i++;

Putting it altogether: 

   
      for (var i = 0; i < 5; i++) {
         code here...
      }
For more information on loops click here.

Arrays

      var names = ["Heather", "John", "Abigail", "Moroni"];
   
      var names = new Array("Heather", "John", "Abigail", "Moroni");

If we want to access the value of "Moroni" we would do it like this: 

      names[3];

We could change "Moroni" to "Joseph" like this: 

      names[3] = "Joseph";

Now the array looks like this: 

      var names = ["Heather", "John", "Abigail", "Joseph"];

For loops are often used to loop through arrays. JavaScript has many built-in methods for interacting with array.length returning the legnth of the array. The name of your array will go in place of array. An interation through this array might look like this: 

      for(var i = 0; i < names.length; i++) {
         studentsHere += names[i];
         return studentsHere;
      }
